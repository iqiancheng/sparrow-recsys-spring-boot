# sparrow-recsys-spring-boot
Spring Boot 版本的SparrowRecSys，支持 Docker 一键运行。

## Overview
![](docs/related.png)

## Usage
```shell
mvn clean package # 先清理编译结果，再编译
docker-compose up -d --build --force-recreate # 启动容器
```
